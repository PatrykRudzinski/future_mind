# Zadanie rekrutacyjne – Aplikacja filmowa (OMDb API)

Wykorzystując publiczne API OMDb API - The Open Movie Database stwórz aplikację pozwalającą na wyszukiwanie filmów oraz przeglądanie szczegółów wybranego tytułu.

## Funkcjonalności

1. Strona główna

- formularz wyszukiwania filmów
- możliwość filtrowania po:
  - roku premiery
  - typie
- lista wyników wyszukiwania
- paginacja:
  - może być klasyczna
  - lub infinite scroll

2. Widok szczegółów filmu

- wyświetlenie informacji o filmie (np. tytuł, opis, gatunek, rok, ocena, poster itp.)

3. Ulubione

- możliwość dodawania i usuwania filmów z listy „ulubionych”
- ulubione powinny być trwałe po odświeżeniu strony

## Wymagania techniczne

- aplikacja może być zbudowana jako SPA lub jako aplikacja SSR (np. Next.js)
- aplikacja powinna:
  - obsługiwać błędy z API
- być responsywna
- spełniać podstawowe wymogi WCAG
- być zoptymalizowana pod kątem SEO
- mile widziane testy jednostkowe.

> Prosimy o umieszczenie aplikacji w dowolnym repozytorium, README z instrukcją uruchomienia, krótki opis zależności. Będziemy zwracać uwagę na dobór narzędzi i sposób ich wykorzystania, type safety, czytelny oraz uporządkowany kod.
